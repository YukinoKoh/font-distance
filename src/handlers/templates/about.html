About Font Distance

This web app helps you to find similar fonts of your choice, looking at font balances and weight.
I hope this project gives you some idea when you choose fonts for a project that involves Japanese writing and English writing. I myself find it not easy to set fonts for projects involving different languages. I think it is because those font families have in a way completely different evolving histories. I appreciate fonts designed for such situation, Google Noto Fonts, or Adobe Source Han Serif. At the same time I think possible combinations of fonts vary, as people write various things.

For this app, I chose fonts in this way:

  • I chose serif (Gothic in Japanese), sans serif (Mincho in Japanese), or monospace, assuming to set fonts for body or long text.
  • If the font is available as open source, Typekit, or commonly installed in Mac and Windows to make sure users can see the same shape of the fonts.
  • I started with fonts that I found some way related to texts by Akira Kobayashi, Ellen Lupton, or Stephen Coles.

With this app you can see overall font relationships in Font Distance, test typing in Font Test, and look up similar fonts in Similar Fonts in Japanese, or in Similar Fonts in English. Due to data size, these fonts do not support open type function.

To calculate 'font distance', this app compares the parameters of each font, for example, line width, character width, ratio of x-height to cap height, or ratio of Kanji volume to Hiragana volume.

I started this project after inspired by Font Map, which supported by machine learning technique. My app is not directly supported by machine learning, rather it is a hand tweaked project, but several concepts and techniques of machine learning gave me hint while I was thinking its algorithm.

Hope you enjoy it!
