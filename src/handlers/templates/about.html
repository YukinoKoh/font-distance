About Font Distance

This web app helps you to find similar fonts to your choice, looking at font balances and weight.
I hope this project help you when you are choosing fonts for a project that involves Japanese writing and English writing. I myself find it not wasy to set fonts for such projects, because a challenge in the process is balancing 2 different font families which have in a way completely different evolving histories. For this reason I appreciate fonts designed for contents with various languages, such as Goolge Noto Fonts, or Adobe Source Han Serif. At the same time I think possible combinations of fonts vary, as people write various things.

With this app you can see overall font relationships in Font Distance, test typing with selected fonts in Font Test, and look up similar fonts in Similar Fonts in Japanese, or in Similar Fonts in English. Due to data size, these fonts do not support open type function.

To calculate 'font distance', this app compare params of each fonts, for example, line width, character width, ratio of x-height to caps height, which I added individually.

I started this project after inspired by Font Map, which supprted by machine learning technique. My app is not directly supported by machine learning, rather it is hand tweeked project, but several concept and techniques of machine learning gave me hint while I was thinking its algorithm.

Hope you enjoy it!
